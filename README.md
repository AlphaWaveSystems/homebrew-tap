# homebrew-tap

Homebrew tap for [FlutterProbe](https://flutterprobe.dev) — E2E testing CLI for Flutter apps.

## Install

```bash
brew tap AlphaWaveSystems/tap
brew install probe
```

## Upgrade

```bash
brew upgrade probe
```

## Usage

```bash
probe --version
probe test tests/ --device <device-id>
```

Full documentation at [flutterprobe.dev](https://flutterprobe.dev).
